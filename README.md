# Fetal Health Detection System

This is an AI-based web application that detects fetal health conditions from medical scan images.

## Technologies Used
- Python
- Flask
- YOLO Deep Learning
- HTML, CSS
- SQLite Database

## Features
- Upload fetal scan image
- AI model analyzes the image
- Displays health status
- Stores prediction history

## Purpose
To support early detection of fetal health issues using artificial intelligence.
